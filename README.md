# ForgeManager

<p align="center">

<img src="docs/images/logo.png" width="180" alt="ForgeManager Logo">

### Modern Windows Server Management for Arma Reforger

A powerful all-in-one Windows 10/11 desktop application for installing, configuring, monitoring, updating, and managing **Arma Reforger Dedicated Servers**.

Designed to eliminate command-line setup while providing powerful administration tools, live monitoring, workshop integration, and production-ready server management.

</p>

---

# Features

## 🚀 One-Click Server Installation

- Automatic SteamCMD download
- Automatic SteamCMD installation
- Install Stable or Experimental server builds
- Install to any drive or folder
- Verify game files
- Update server with one click
- Live installation progress
- Download status monitoring

---

## 🎮 Server Management

- Start Server
- Stop Server
- Restart Server
- Kill Server
- Detect already running server instances
- Attach ForgeManager to externally launched servers
- Automatic executable detection
- Automatic server status detection
- Automatic crash detection
- Restart after crash (optional)

---

## ⚙ Server Configuration

Complete visual editor for:

- Server Name
- Password
- Admin Password
- Ports
- Max Players
- Public/Private
- Scenario
- Mods
- Missions
- BattlEye
- RCON (future)
- Logging
- Performance options

No manual JSON editing required.

# Why ForgeManager?

Traditional dedicated server management often requires:

- SteamCMD
- JSON editing
- Batch files
- Command Prompt
- Manual updates
- Manual mod installation

ForgeManager replaces all of that with a modern graphical interface while still exposing advanced options for experienced administrators.
